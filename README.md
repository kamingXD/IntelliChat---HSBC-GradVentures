# IntelliChat

IntelliChat is a smart, conversational financial assistant designed to provide personalized advice. Built with a modern tech stack, it offers a fast, responsive, and intuitive user experience.

## ‚ú® Features

- **Conversational AI**: Powered by Google's Gemini model via Genkit for natural and relevant financial advice.
- **Real-time Streaming**: AI responses are streamed in real-time, creating a fluid and engaging conversation.
- **Chat History**: Automatically saves conversations, which can be reviewed or deleted from the sidebar.
- **Product-Focused Chats**: Users can start conversations based on specific financial products like "Loans" or "Credit Cards".
- **Responsive Design**: A clean and modern UI built with Next.js, React, ShadCN UI, and Tailwind CSS.

## üöÄ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (with App Router)
- **UI Library**: [React](https://react.dev/)

- **Components**: [ShadCN UI](https://ui.shadcn.com/)
- **Generative AI**: [Google Gemini](https://deepmind.google/technologies/gemini/)
- **AI Orchestration**: [Genkit](https://firebase.google.com/docs/genkit)

To run the application locally, you will first need to install the dependencies.

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

Clone the repository and install the required packages:

```bash
npm install
```

### Running the Development Server

To start the development server, run:

```bash
npm run dev
```

The application will be available at [http://localhost:9002](http://localhost:9002).

To get started, take a look at the main page in `src/app/page.tsx`.
