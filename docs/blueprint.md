# **App Name**: IntelliChat

## Core Features:

- Chat Interface: A standard chat interface for mobile, displaying messages in a conversational format.
- LLM Integration: Connects to the OpenAI LLM using the user provided API configuration and code.
- Message Streaming: Implements a streaming approach to show responses from the LLM model.
- Intelligent Suggestion Tool: Use a tool that detects when to use the LLM reasoning to suggest multiple-choice responses.
- Input field: Ask something.
- Button for sending message: Triangle-shaped red button on the right side of the message input box for sending a message
- Chat menu: Menu to give options for interacting with a Chat
- Product Type Selection: Page to select product types: loans, insurance products, credit cards, investments and saving accounts, for personalized financial advice.
- Conversation History Management: Allow users to add, delete, and navigate previous chatrooms.

## Style Guidelines:

- Primary color: Light grayish-red (#E2726B), suggesting institutional trust without aggression. Referencing HSBC style.
- Background color: Very light grayish-red (#F9E2E0), near-white background provides visual clarity for the chat interface. Referencing HSBC style.
- Accent color: Pale orange-red (#F0954A) for send button. Referencing HSBC style.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a clean, readable appearance suitable for both headlines and body text. Referencing HSBC style.
- Simple, outline-style icons for menu options to maintain a clean interface. Referencing HSBC style.
- Classic chat layout with message bubbles, input field at the bottom, and clear separation of messages. Referencing HSBC style.
- Subtle animations for message transitions and loading indicators to provide feedback without distraction. Referencing HSBC style.
